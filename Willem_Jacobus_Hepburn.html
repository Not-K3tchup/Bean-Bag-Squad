Evolution Of The Internet

Web 2.0 - 4.0

In the last two decades, a lot has happened in the world of the internet and related technologies. Since the beginning of the web, four generations of the web have been introduced: web 1.0 as a web of cognition, web 2.0 as a web of communication, web 3.0 as a web of collaboration, and web 4.0 as a web of integration (Aghaei, et al., 2012). The era (1990–2000) when a user's function was confined to reading information provided by content creators is known as Web 1.0, also known as Syntactic web read only web (Madurai, 2018).

. . .

Dale Dougherty defined Web 2.0 as a read-write web in 2004, Web 2.0 technologies enable massive worldwide populations with common interests in social interactions to be assembled and managed (Aghaei, et al., 2012). Web 3.0, often known as the semantic web, aims to delegate human duties and choices to machines by making web information machine-readable. Web 3.0 encompasses two key platforms: semantic technologies and the social computing environment (Madurai, 2018). Web 4.0 will be a read-write-execution-concurrency web with intelligent interactions, but no precise definition has been given. Web 4.0 is also known as the symbiotic web, because it allows humans and machines to work together in harmony (Aghaei, et al., 2012).
