Client - Server Side

Client side engineering is where developers uses coding skills to develop visual appealing web sites and web applications. These web pages and web applications are displayed in browsers on computers and smartphones. Users interact with these pages and web applications to perform functions like learning, research or information processing.

Client-side means that the processing takes place on the user's (the client's) computer or smart phone. Server-side means that the processing takes place on a web server. This is also called client-server architecture.

Client side developers are also called front-end developers. Common scripting languages for client side development is Javascript, HTML and CSS. Just like what we are doing currently at WPR181.
